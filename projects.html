Configuración necesaria para sincronización

IMPORTANTE: Para ver tus proyectos en todos los dispositivos, necesitas configurar un token de GitHub.

Sigue estos pasos para configurar la sincronización:

  1. Ve a GitHub Personal Access Tokens
  2. Crea un nuevo token con permisos gist
  3. Pega el token en el campo siguiente
  4. Haz clic en "Guardar Token"
Este token se guardará localmente en tu navegador.
